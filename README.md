# Microservices CI/CD Project

## Project Description

This project demonstrates the implementation of CI/CD pipelines using GitHub Actions, Docker, Terraform and AWS OIDC authentication.

The repository contains Java microservices and several GitHub Actions workflows for automation, testing, Docker image publishing and infrastructure validation.
